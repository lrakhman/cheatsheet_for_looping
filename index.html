Looping in Ruby and JavaScript

Below is a comparison of the different looping constructs available in Ruby and JavaScript.

Looping construct Ruby JavaScript
Loop

Ruby has a basic "loop". An explicit "break" must be added to prevent an infinite loop.

Example: i = 20 loop do print i break if i <= 0 i -= 1 end

JavaScript does not have a basic "loop" construct, but the same result can be achieved with a "while(true)" loop.

Example: var i = 20; while( true ) { console.log(i) if (i <= 0) { break } i -= 1; }
While/Until

Ruby has both a "while" loop and an "until" loop. Logically, "while(true)" is equivalent to "until(false)". Below are some examples.

While example #1: counter = 1 while counter < 11 puts counter counter += 1 end Until example #1: counter = 1 until counter > 10 puts counter counter += 1 end While example #2: i = 0 num = 5 begin puts("Inside the loop i = #{i}" ) i +=1 end while i <= num Until example #2: i = 0 num = 5 begin puts("Inside the loop i = #{i}" ) i +=1; end until i > num

JavaScript has a "while" loop but no "until" loop. However, "until" is logically equivalent to "while(false)", so "until" loops can easily be imitated in JavaScript.

While example #1: var counter = 1 while (counter < 11) { console.log(counter); counter += 1; } While example #2: var i = 0; var num = 5; do { console.log("Inside the loop i = " + i); i += 1; } while (i <= num);
For

Ruby has a basic "for" loop that is used with collections (ranges, array, hashes).

For example: for num in 1..20 puts num end

JavaScript has two types of "for" loops: a basic "for" loop with collections, like Ruby's "for" loop, and a "for...in" loop that is used to loop through properties of objects.

For example: for (var num = 1; num < 21; i++) { console.log(num); } For...in example: var myObj = {a: 1, b: 2, c: 3}, myKeys = []; for (var property in myObj) { myKeys.push(property); } myKeys; //['a','b','c'];
