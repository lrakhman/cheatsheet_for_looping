Looping in Ruby & JavaScript

Below is a comparison of the different looping constructs available in Ruby and JavaScript.

Ruby JavaScript

LOOP DO

Ruby has a basic "loop". An explicit "break" must be added to prevent an infinite loop.

EXAMPLE:

i = 20
loop do
  print i
  break if i <= 0
  i -= 1
end

LOOP DON'T

JavaScript does not have a basic "loop" construct, but the same result can be achieved with a "while(true)" loop.

EXAMPLE:

var i = 20;
while( true ) {
  console.log(i)
  if (i <= 0) {
    break
  }
  i -= 1;
}

WHILE / UNTIL

Ruby has both a "while" loop and an "until" loop. Logically, "while(true)" is equivalent to "until(false)". Below are some examples.

'WHILE' EXAMPLE #1:

counter = 1
while counter < 11
  puts counter
  counter += 1
end

'UNTIL' EXAMPLE #1:

counter = 1
until counter > 10
  puts counter
  counter += 1
end

'WHILE' EXAMPLE #2:

i = 0
num = 5
begin
   puts("Inside the loop i = #{i}" )
   i +=1
end while i <= num

'UNTIL' EXAMPLE #2:

i = 0
num = 5
begin
   puts("Inside the loop i = #{i}" )
   i +=1;
end until i > num

WHILE, NOT UNTIL

JavaScript has a "while" loop but no "until" loop. However, "until" is logically equivalent to "while(false)", so "until" loops can easily be imitated in JavaScript.

'WHILE' EXAMPLE #1:

var counter = 1
while (counter < 11) {
  console.log(counter);
  counter += 1;
}

'WHILE' EXAMPLE #2:

var i = 0;
var num = 5;
do {
  console.log("Inside the loop i = " + i);
  i += 1;
} while (i <= num);

FOR

Ruby has a basic "for" loop that is used with collections (ranges, array, hashes).

FOR EXAMPLE:

for num in 1..20
  puts num
end

FOR and FOR..IN

JavaScript has two types of "for" loops: a basic "for" loop that is used with collections, like Ruby's "for" loop, and a "for...in" loop that is used to loop through properties of objects.

'FOR' EXAMPLE:

for (var num = 1; num < 21; i++) {
    console.log(num);
}

'FOR...IN' EXAMPLE:

var myObj = {a: 1, b: 2, c: 3};
var myKeys = [];
             
for (var property in myObj) {
    myKeys.push(property);
}
 
myKeys; //['a','b','c'];
